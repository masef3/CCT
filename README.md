### CCT

Very simple program that parses *folded* file and stores it in calling context tree data structure.
Also exports and copies stored data into new folded file.

*BASIC GNU TOOLS REQUIRED*

## Linux build:

To simply start using the program just use `make`.

To add the binary to local path use `make install`.

To clean directory `make clean`.

To rebuild program `make rebuild`.

To run the program on a file:

```
./cct filename.folded
```
